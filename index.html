We have worked on

INDUSTRIAL CONTROLLER

An industrial controller built around the Raspberry Pi Compute Module, designed to meet demanding automation and control requirements. The system integrates isolated power supplies, a backup UPS for reliability, and multiple industrial-grade interfaces including HDMI, USB, Ethernet, CAN, and RS485.

STM32 SoM

A compact and high-performance System on Module built around the STM32MP1, featuring external SDRAM for extended memory capacity, onboard eMMC for storage, CAN Bus for industrial communication and an Ethernet controller for network connectivity.

AUTOMOTIVE HUB

It manages a USB 3.1 upstream link and distributes it to USB 3.1, dual USB 2.0 ports, Gigabit Ethernet, and RS485. The module streamlines integration by combining multiple interfaces into a single reliable solution.

RFID/NFC CRYPTO DEV KIT

development kit built around a n RFID/NFC-enabled chip paired with a microcontroller that provides integrated cryptographic handling. The kit is tailored for applications in the cryptocurrency market.

HIGH VOLTAGE INDUSTRIAL CONTROLLER

This industrial-grade controller is designed to serve as the main control unit for high-voltage equipment, as well as RS485 and CAN-based devices.

INDUCTIVE DEV KIT

A dedicated hardware platform created to evaluate the performance and reliability of inductive keyboard technologies.
